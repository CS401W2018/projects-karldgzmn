Welcome

Answer the following questions:

1. What is the difference between block tags and inline tags.

Block tags always start on a new line. Inline tags Do not start on a new line

2. What is the margin, border, and padding?

It defines the distance between the elements. Border is the surrounding outside the elements. Padding is the space between.

3. What are units of measurement you can use in CSS?

px, pt, cm, mm, in

4. When would you use a figure tag?

to provide a caption or description of the figure.
